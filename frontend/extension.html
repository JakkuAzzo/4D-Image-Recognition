Install the Identity Protection Extension

This project includes a reference browser extension under tools/identity_protection/browser_extension you can load unpacked for development.

Chromium (Chrome/Edge/Brave)

  1. Open chrome://extensions or edge://extensions
  2. Enable Developer Mode
  3. Click "Load unpacked" and select the tools/identity_protection/browser_extension folder.

Firefox

  1. Open about:debugging#/runtime/this-firefox
  2. Click "Load Temporary Add-onâ€¦" and select the manifest.json inside tools/identity_protection/browser_extension.

After installing, return to Dual Rig and the button will switch to "Save Config to Extension" automatically.
